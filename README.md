# ecommerce-analytics
Ecommerce analytics project analyzing revenue trends, order behavior, and business metrics using Python (pandas) and SQL

## Project Overview

This project analyzes ecommerce order data to identify revenue patterns, order behavior, and business performance trends.

The goal is to practice SQL, Python (pandas), and business-focused analytics using an ecommerce dataset.
